# HR Employee Attrition Analysis | SQL + Power BI

Completed: June 20, 2026  
Tools: SQL, Power BI, DAX, Data Modeling

## Business Problem
Analyze attrition patterns for 1,470 employees to identify high-risk departments and roles.

## Key Insights
- Sales Representative role: 39.76% attrition vs company avg 16.12%
- Sales Dept paradox: Highest salary 7K but 20.6% attrition vs R&D 13.9%
- Built a data model with relationships between KPI tables
